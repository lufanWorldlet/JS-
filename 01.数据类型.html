数据类型指的是字面量的类型 在JS中一共有六种数据类型 （一：基本数据类型） String 字符串 var str =“hello” \双引号表示 转义字符 Number 数值 在js中所有的数值都是Number类型， 包括整数和浮点数 可以使用一个运算符typeof 来检查一个变量的类型 语法typeof变量 例子： var a=123 console.log(typeof a) 输出： "number" var a="123" console.log(typeof a) 输出： "String" js中可以表示的数字的最大值 console.log(Number.MAX_VALUE); 如果使用Number表示的数字超过了最大值，则会返回一个Infinity 表示正无穷 NaN 表示不是一个数字 如果使用JS进行浮点元素，可能得到一个不精确的结果 Boolean 布尔值 Null 空值 Undefined 未定义 （二：引用数据类型） Object 对象
